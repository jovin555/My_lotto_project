# Lotto Max Analysis Project

Analyze Lotto Max winning numbers and generate statistical predictions.

## Features

- 📊 **Frequency Analysis**: Identify which numbers appear most often
- 📈 **Statistical Summary**: Mean, median, standard deviation, range
- 🔮 **Predictions**: Based on historical frequency patterns
- 📉 **Visualization**: Charts and graphs of number distributions

## Setup

```bash
# Install dependencies
pip install -r requirements.txt

# Run the analyzer
python lotto_analyzer.py
```

## Output

The analyzer generates:
- Console report with statistical analysis
- `frequency_analysis.csv` - Number frequencies for further analysis

## Important Note

**Lottery drawings are random events.** Each number has an equal probability of being drawn in each draw. Historical frequency analysis is provided for statistical interest only and should not be used for gambling decisions.
